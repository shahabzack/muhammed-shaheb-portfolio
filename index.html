XAI_TraderX – Stock Trading Assistant with GRU + RAG

XAI_TraderX predicts next-day closing prices for Axis Bank and Reliance using a GRU model. Users can simulate trades without real money, ask about past trades and profits via a RAG-powered chatbot backed by ChromaDB, and view model performance over the last 7, 14, and 30 days. It also generates trade summary reports for any period (weekly, monthly, or custom) based on user queries. Built with FastAPI and Streamlit, and hosted on AWS.

Knowra – RAG Chatbot for PDF Understanding

Knowra is an intelligent Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions based on their content. It uses FAISS for semantic search, LangChain for pipeline integration, and Gemini Pro for natural language answers. Built with FastAPI and Streamlit, it features chat memory, custom UI, and modal-based PDF upload. The full version is deployed on Render, while a lightweight version is available on Hugging Face Spaces.
